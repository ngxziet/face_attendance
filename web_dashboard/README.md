# Face Attendance System - Web Dashboard

React web dashboard for managing the Face Attendance System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The dashboard will be available at http://localhost:3000

## Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

## Features

- User management (CRUD operations)
- Face enrollment via webcam
- Real-time attendance statistics
- Attendance reports with CSV/Excel export
- WebSocket integration for live updates

## Default Admin Account

You need to create an admin account in the backend database first.
